# Bejeweled Twist Game

A browser-based match-3 puzzle game inspired by Bejeweled Twist, featuring special gems with powerful effects.

## Features

- **8x8 Grid** with 6 colorful gem types
- **Twist Mechanic** - Click and drag to rotate 2x2 groups of gems
- **Match 3+** - Match gems horizontally or vertically to score points

- **Chain Reactions** - Cascading matches for massive combos
- **Move Counter** - Complete objectives within 30 moves
- **Smooth Animations** - Polished visual feedback and effects

## How to Play

1. Click and drag on the game board to rotate a 2x2 group of gems
2. Create matches of 3 or more gems of the same color
3. Match 4+ gems to create special gems with powerful effects
4. Activate special gems by including them in a match
5. Try to get the highest score before running out of moves!

## Play Now

Simply open `index.html` in your web browser to start playing!
